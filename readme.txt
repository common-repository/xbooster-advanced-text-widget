Easy to use Advanced Text Widget. It is possible to show/hide widget on any pages/posts/categories you want including homepage.

== Description ==

xBooster Advanced Text Widget is extremely useful if you have deep  category structure and different page templates for different type of posts.

Do you need show some sidebar banners on some pages and hide on some?

Do you want to show a custom HTML on specific pages?

Do you want to hide a custom HTML on specific posts etc?

This is the plugin you are looking for.

Please visit http://www.allthemesnulled.com see our plugins in action.

Detailed documentation is <a href="http://allthemesnulled.com/xbooster-advanced-text-widget/">here</a>

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Your new widget is in Apperance > Widgets menu
